# World Cup Sentiment Tracker

## Goal

Build a real-time system that tracks public sentiment during World Cup matches and analyzes whether fan sentiment correlates with actual match performance.

## MVP

Input:
- Sample football tweets stored in CSV

Processing:
- Run sentiment analysis
- Store sentiment scores

Output:
- Sentiment timeline
- Team sentiment comparison

## Future Features

- Live tweet collection
- Player-level sentiment
- Event detection
- Fine-tuned sentiment model
- Real-time dashboard
- Correlation analysis
